Datenschutzhinweis

Diese Website dient der Präsentation privater 3D-Druck Projekte und enthält keine direkte Bestellabwicklung oder Registrierung.

Kontaktaufnahme

Wenn du über die Anfrage-Seite Kontakt aufnimmst, wird eine E-Mail an die angegebene Adresse erstellt. Die von dir eingegebenen Daten werden dabei nur zur Bearbeitung deiner Anfrage genutzt.

Keine direkte Speicherung über Formulare

Die Anfrage auf dieser Website wird nicht auf einem Server gespeichert. Die Seite erstellt lediglich einen sogenannten „mailto“-Link, über den dein eigenes Mailprogramm geöffnet wird.

Externe Links

Auf dieser Website können Links zu externen Plattformen enthalten sein, zum Beispiel MakerWorld, Etsy oder andere Seiten. Für Inhalte und Datenschutz dieser externen Seiten ist jeweils der dortige Betreiber verantwortlich.

Technische Daten

Beim Aufruf einer Website können durch den Hosting-Anbieter technisch notwendige Daten verarbeitet werden, etwa IP-Adresse, Zeitpunkt des Zugriffs oder Browserinformationen. Darauf besteht seitens dieser Website kein gesonderter Einfluss.

Änderungen

Dieser Hinweis kann angepasst werden, wenn sich Inhalte oder technische Funktionen der Website ändern.
